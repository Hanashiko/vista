# 🧭 Quest App

**Quest App** — це веб-додаток для створення, проходження та оцінювання квестів. Складається з бекенду на Flask та фронтенду на React.

## 📂 Структура проєкту

```
.
├── backend/    # Flask REST API (Python)
├── frontend/   # React SPA (JavaScript)
└── README.md   # Поточний файл
```

- 🔧 **Backend** — розташований у [./backend](./backend), реалізує API для квестів, задач, статистики, прогресу та рейтингів.  
  ➡ Документація: [backend/README.md](./backend/README.md)

- 🎨 **Frontend** — знаходиться у [./frontend](./frontend), реалізований на React.  
  ➡ Документація: [frontend/README.md](./frontend/README.md)

---

## 🚀 Швидкий старт

### 🔙 Backend

```bash
cd backend
pip install -r requirements.txt
flask run
```

> Детальніше в [backend/README.md](./backend/README.md)

---

### 🔜 Frontend

```bash
cd frontend
npm install
npm start
```

> Детальніше в [frontend/README.md](./frontend/README.md)
